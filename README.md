# MPDX

You'll need to store in your environment variables a `FONTAWESOME_NPM_AUTH_TOKEN`. You can request this by sending an email to [apps@cru.org](mailto:apps@cru.org). This will allow you to pull the font-awesome-pro npm package used in this project.

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/zeit/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
